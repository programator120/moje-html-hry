# moje-html-hry

Jednoduchá stolová hra **Dostihy** pripravená na okamžité otvorenie v prehliadači. Hra obsahuje ľahké SVG sprity koníkov a prepínanie textov medzi slovenčinou, češtinou a angličtinou.

## Ako hrať
1. Otvor `index.html` v modernom prehliadači (nie je potrebný žiadny server ani build nástroje).
2. Vyber počet jazdcov (2–4) a jazyk rozhrania.
3. Stlač **Začať hru** a následne **Hodiť kockou** pre aktívneho jazdca.
4. Prvý, kto dokončí tri kolá, vyhráva dostihy. Všetky hody sa zapisujú do denníka.

## Čo obsahuje
- Responzívny layout so zvýraznenými úsekmi trate.
- SVG sprity štyroch koníkov (`sprites/horse-*.svg`).
- Preklady ovládania aj hlásení v EN/SK/CZ.
- Ľahká logika na postup jazdcov, počítanie kôl a zápis udalostí.
